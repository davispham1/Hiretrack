# Job Application Tracker

A simple job application tracker built to help organize applications, track statuses, and view progress in a cleaner way.

## Features

- Add and manage job applications
- Track statuses like Applied, OA, Interview, and Offer
- View a status ratio chart
- See application activity in a heatmap
- Filter and browse saved applications

## Tech Stack

- React
- Vite
- JavaScript
- CSS

## Getting Started

### 1. Clone the repo


git clone https://github.com/davispham1/Internship-Tracker.git
npm install
npm run dev
